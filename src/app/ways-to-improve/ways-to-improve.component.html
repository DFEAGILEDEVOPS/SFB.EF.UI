Tools to improve your school's efficiency

There are several tools from the DfE and third-party providers we recommend as useful to help schools see how they can help improve their school's efficiency.

  • Financial planning checklist

    The DfE provides tools and information to help schools with financial planning and benchmarking.

    Use the financial planning checklist

  • Ask the right questions

    The Top 10 checks for financial planning explains what to look at when you’re preparing to question and challenge school budgets. It’s aimed at governors but useful for everyone.

    Use the top 10 financial planning checks

  • Evaluate different strategies for improving learning

    The Education Endowment Foundation has a toolkit that provides evidence-based summaries about different strategies for improving learning. You can see how much each strategy costs and how much impact it has.

    Use the toolkit
